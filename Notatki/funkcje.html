1// Funkcja to nazwany blok kodu gotowy do wielokrotnego użycia
2function przywitaj(imię) {
3 return `Cześć, ${imię}!`;
4}
5
6// Wywołanie — podajemy argument w nawiasach
7przywitaj("Ania"); // → "Cześć, Ania!"
Słowo kluczowe function deklaruje funkcję. przywitaj to nazwa. imię w nawiasie to parametr — dane wejściowe. return zwraca wartość do wywołującego.
Wyobraź sobie funkcję jak przepis kuchenny — zapisujesz go raz, używasz dowolną ilość razy z różnymi składnikami.

deklaracja

1function suma(
2 a, b) {
3 return a+b;
4}
Hoistowana — działa nawet przed miejscem definicji.

wyrażenie

1const suma =
2 function(
3 a, b) {
4 return a+b;
5};
Przypisana do zmiennej. Nie jest hoistowana.

funkcja strzałkowa

1const suma =
2 (a, b)
3 => a + b;
Skrócony zapis. Jedno wyrażenie = auto return.
Wszystkie trzy robią to samo. Różnią się składnią i zachowaniem this. Funkcje Strzałkowe dominują w nowoczesnym JS.
